# colorConversion 
This repository is a simple BGR2YUV demo. It gets BGR channels by loading a BMP image. 

## Support:
1. Load BMP 
2. Save BMP 
3. BGR888ToYUVI444 
4. YUVI444ToBGR888 
5. YUVI444ToYUV420 
6. YU12 
7. YV12 
8. NV12 
9. NV21 

## Modify:  
1. Support for the conversion of odd resolution images
